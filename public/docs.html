Tutorial

Get started with WingJS

WingJS is a CSS framework based on a simple and easy-to-use system based on components. We provide you a large pane of mordern and fully-responsive components sets, including the most used basic atoms.

Installation

Start the installation by cloning the repository from GitHub. 

git clone https://github.com/mathcovax/Challenge-Stack.git

Docker

Build the project environment with Docker. 

docker-compose up --d

Run the framework

Start a development webpack server to run the project on a localhost. 

npm run dev
